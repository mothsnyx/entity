Quick Info
Welcome to The Entity Dashboard!

Use this dashboard to manage all aspects of the bot:

  • Profiles: View all created character profiles
  • Realms: Add, edit, or remove game realms
  • Shop Items: Manage items available for purchase
  • Trial Messages: Customize messages for Killer and Survivor trials
  • Minigames: Configure loot tables for hunting, fishing, and scavenging

All changes are saved directly to the database and will be reflected in the Discord bot immediately.
