Profile Management
How to Manage Profiles

Profiles are created and managed through Discord slash commands:

  • /create [name] [role] - Create a new profile
  • /editname [name] [new_name] - Change profile name
  • /editrole [name] [new_role] - Change profile role
  • /deleteprofile [name] - Delete a profile
  • /profile [name] - View profile details
